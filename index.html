How to refresh the data

Data refreshes automatically every day at 07:00 BST via a scheduled GitHub Action — no manual steps needed.

To trigger an immediate refresh, go to the Actions tab on GitHub, select Scrape Physio Availability, and click Run workflow.

To run locally:

  1. Install dependencies once:
    pip install -r requirements.txt
  2. Run the scraper:
    python scraper.py
  3. Commit and push data/sites.json to update the live site.
