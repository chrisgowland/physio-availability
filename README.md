# Nuffield Health Physio Availability Dashboard

A web dashboard showing physio availability across all Nuffield Health gym and hospital sites in the UK (~135 sites).

## What it shows

For each site:
- **Physio count** — number of physios (tap to see names + online/in-clinic status)
- **Online booking** — whether the site accepts online bookings
- **Next available** — date and time of the next bookable appointment
- **Slots (4 wks)** — total appointment slots available in the next 4 weeks
- **Links** — direct booking link and Nuffield Health site profile

## Quick start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Scrape the data

```bash
python scraper.py
```

A full scrape of all ~135 sites takes roughly **3–5 minutes** (5 parallel workers, gentle rate limiting).

### 3. Serve the website

The dashboard uses `fetch("data/sites.json")` which requires a web server (not a bare `file://` URL).

**Option A — Python built-in server (simplest):**
```bash
python -m http.server 8080
# then open http://localhost:8080 in your browser
```

**Option B — VS Code Live Server extension:** right-click `index.html` → "Open with Live Server"

**Option C — Deploy:** copy all files to any static web host (GitHub Pages, Netlify, Vercel, etc.)

## Scraper options

| Flag | Description |
|------|-------------|
| `--limit N` | Scrape only the first N sites (useful for testing) |
| `--slug NAME` | Scrape a single site by slug, e.g. `--slug cannock` |
| `--workers N` | Number of parallel threads (default: 5) |
| `--output PATH` | Output file path (default: `data/sites.json`) |
| `--headless` | Use Playwright to render JavaScript on location pages (see below) |

### Getting full physio names (optional)

Physio profile names on Nuffield Health's location pages are rendered by JavaScript.
Without `--headless`, the scraper only lists physios who have live online booking slots.
With `--headless`, Playwright renders the full page and extracts all physio names.

```bash
pip install playwright
playwright install chromium
python scraper.py --headless
```

## Data sources

| Data | Source |
|------|--------|
| Site list (slugs) | `nuffieldhealth.com/physiotherapy` |
| Site address, phone | `nuffieldhealth.com/physiotherapy/[slug]` |
| Physio names (partial) | `nuffieldhealth.com/physiotherapy/[slug]` — requires JS rendering |
| Availability (slots, next appointment) | `book.nuffieldhealth.com/physio/appointments/[slug]` |
| Physio IDs (booking system internal) | Embedded JSON in the booking page |

## Profile links

Nuffield Health does not provide individual per-physio URLs. Each physio's profile link goes to the site's Nuffield Health page (`nuffieldhealth.com/physiotherapy/[slug]`), where their expandable bio section appears. The booking page link (`book.nuffieldhealth.com/physio/appointments/[slug]`) allows selecting a specific physio from a dropdown.
